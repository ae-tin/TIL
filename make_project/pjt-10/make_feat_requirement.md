## 프로젝트 세부 설계

### Vue 프로젝트 세부 설계 과정

#### 라우팅 설계

- 기능 구현에 필요한 최소한의 Route Tree 구상

- 중첩 라우팅(Nested) vs 단일 라우팅 결정
  
  - URL 에서의 중첩된 관계를 표현하는 관점

- 라우터 가드 전략 세우기
  
  - 전역 가드
    
    - 모든 라우트 전환에 적용되는 가드
  
  - 라우터 가드
    
    - 특정 라우트만 적용되는 가드
  
  - 컴포넌트 가드
    
    - 컴포넌트 내에서만 적용되는 가드

- RESTful URL 설계
  
  - 자원의 식별
    
    - URI
  
  - 자원의 행위
    
    - HTTP Methods
  
  - 자원의 표현
    
    - json data(궁극적으로 표현되는 데이터 결과물)

- 스타일 가이드 문서(Figma)

### 문서화

- 일정 계획 수립
  
  - WBS(Work Breakdown Structure)
    
    - 프로젝트를 큰 목표부터 작은 작업 단위까지 계층적으로 나누어 정리한 구조
    
    - 프로젝트 범위와 목표를 정하고, 각 단계의 결과물을 나열하며 작업을 할당
    
    - 담당자 배정과 일정 관리를 위해 사용
  
  - Gantt Chart
    
    - 프로젝트 일정을 타임 라인으로 나열하여, 프로젝트 기간, 진행률을 나타내는 표현 방법
    
    - WBS와 함께 작성하여 프로젝트 진행사항을 한눈에 파악할 수 있음
